# QiitaNightMode
ChromeのQiitaナイトモード拡張機能です。

と、言っても500円も払えないのでストアでの公開はやめにしました。

使いたい人はこのリポジトリをダウンロードして解凍してください。
chromeの「パッケージ化されていない拡張機能を読み込む」で解答したフォルダの中の`QiitaNightMode.zip`ファイルを開いてください。
